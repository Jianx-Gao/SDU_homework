# PCG Categrioes with Raspberry Zeros

- docs：演示文件
  - 单个音频文件的**数据预处理+特征提取**演示代码

- model：模型文件
  - 神经网络模型代码
  - h5模型
  - tflite模型
  - h5模型转换为tflite模型代码

- raspberry：部署树莓派相关文件
  - 核心代码
  - 相关材料

- result：结果文件
  - 神经网络训练结果文件（.npy）
  - 混淆矩阵代码

- extract_bispectrum.py：提取二阶谱代码

- train_2.py：模型一的代码（二分类，正常/异常）

- train_4.py：模型二的代码（四分类，AS/MS/MR/MVP）
